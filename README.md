# Onitama
# This is a A-level NEA project producing the 1 player functionality for the 2 player game onitama
# the project aims to offer all functionality in a single package, including a graphical interface, computer players with variable difficulty and various quality of life functions such as a review feature, hints and the ability to undo the game file
# this program required a saves file to store saved games
# the program was built for linux on a ubuntu distrobution, however all functionality apart from built in benchmarking functions can be ran on linux
# pre requisites include; numpy, pandas, pygame all other imported libraries are either unecessary for the standard functionality or part of the standard python package
